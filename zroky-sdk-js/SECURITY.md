# Security Policy

Zroky Watch OSS handles sensitive telemetry paths, so please report security issues privately.

## Reporting a vulnerability

Email `security@zroky.ai` with:

- affected repo and version
- impact summary
- reproduction steps or proof of concept
- suggested fix, if known

Please do not open a public issue for suspected vulnerabilities.

## Scope

Security reports may include SDK capture safety, gateway header stripping, redaction, replay artifact verification, auth handling, secret leakage, or unsafe defaults.

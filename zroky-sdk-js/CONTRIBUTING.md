# Contributing

Thanks for helping improve Zroky Watch OSS.

## Scope

External contributions are welcome for the open-source data-plane repositories:

- SDK capture behavior
- Gateway provider compatibility and redaction
- Replay worker protocol safety
- Tests, examples, docs, and CI hardening

The proprietary Zroky backend, dashboard, judge logic, billing, and autonomy engine are not part of this repo.

## Development

1. Fork the repo.
2. Create a small branch.
3. Add focused tests for behavior changes.
4. Run the repo CI commands from the README.
5. Open a pull request with a clear problem statement and verification notes.

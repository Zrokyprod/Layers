# Zroky Build Tracker

Status values: Pending, In Progress, Blocked, Implemented, Verified, Deferred, Deleted

Completion rule:

```text
Verified = code or deletion is complete, focused check passed, and evidence is recorded.
Implemented = code or deletion is complete, but verification is not yet complete.
Blocked = next action needs user decision, external access, or a failing prerequisite.
```
